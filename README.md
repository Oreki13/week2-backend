- DELETE

- PAGINATION

### INSTALATION

- Set .env to your db setting

```
DB_HOST = 'localhost'
DB_USER = '....'
DB_PASSWORD = '....'
DB_DATABASE = '....'
PORT = 8080
NODE_ENV = 'Devploment'
```

- Start Server

```
$ npm start
```
